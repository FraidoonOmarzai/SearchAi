# SearchAi College Project

This project consist of ten apps, where first I trained Machine Learning and Deep Learning model.
After that I deploy them using Flask.

## Applications

- Heart Disease
- Kidney Disease
- Diabetes
- Liver Disease
- Breast Cancer
- Number Detector
- Spam Detector
- Restaurant Review
- Image Classifier
- House Price
